# Decorative Textures

This directory contains decorative texture files for ornate frame materials.

## Required Textures

Place the following texture files in this directory:

### Ornate Pattern
- `ornate_diffuse.jpg` - Base color/diffuse map
- `ornate_normal.jpg` - Normal map for surface detail

## Texture Specifications

- **Format**: JPG or PNG
- **Recommended Resolution**: 1024x1024 or 2048x2048
- **Color Space**: sRGB for diffuse maps
- **Normal Map Format**: OpenGL format (Y+)
- **Pattern**: Ornate patterns, baroque designs, or decorative motifs

## Finding Textures

Free texture resources:
- [Poly Haven](https://polyhaven.com/textures)
- [CC0 Textures](https://cc0textures.com/)
- [Texture Haven](https://texturehaven.com/)
- [3D Textures](https://3dtextures.me/)

## Custom Patterns

You can also create your own decorative patterns using:
- Adobe Photoshop
- GIMP (free alternative)
- Substance Designer
- Blender's texture painting tools

## Usage

These textures are automatically loaded by the `MaterialLibrary` class in `js/components/materialLibrary.js`.
